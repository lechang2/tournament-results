1. Open terminal, direct to vagrant directory
2. type vagrant up turn on VM
3. type vagrant ssh to logon
4. go to the tournament directory
5. type psql
6. type 'create database tournament' to create the database
7. type \i tournament.sql to create the tables, then \c tournament to connect to it
8. type \q to quit psql
9. type python tournament_test.py to test the program
